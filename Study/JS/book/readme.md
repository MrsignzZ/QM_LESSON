- 用闭包来构建<br>
  函数内嵌函数  就会形成作用域的嵌套，形成闭包<br>
  因为闭包内部的函数，会保存对外面函数内部的变量的持久引用，让这些变量的生命周期得以延续<br>
  闭包，函数在运行时，可以引用到定义时的上下文环境内的变量。
